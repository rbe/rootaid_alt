----

== Start

----
cd asciidocserver
CONTENT_ROOT=/home/rootaid/eu.artofcoding.rootaid/asciidoc
nohup node asciidocserver.js --bookroot=${CONTENT_ROOT}/book --blogroot=${CONTENT_ROOT}/blog 1>>asciidocserver.log 2>&1 &
----
