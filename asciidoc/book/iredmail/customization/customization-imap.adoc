=== IMAP/POP Server Dovecot

==== SSL/TLS

[source,bash,linenums]
.Setup SSL certificate
----
ssl_ca = </etc/ssl/certs/mx.1ci.net.ca-bundle
ssl_cert = </etc/ssl/certs/mx.1ci.net.crt
ssl_key = </etc/ssl/private/mx.1ci.net.key
----
